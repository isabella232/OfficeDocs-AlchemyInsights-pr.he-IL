# <a name="change-the-font-size-in-a-message"></a>שינוי גודל הגופן בהודעה

1. אם כבר הקלדת את ההודעה, בחר את הטקסט.
2. בחר את  **סמל** עיצוב מתחת להודעה (זהו הסמל הראשון).
3. בחר את  **סמל גודל**  הגופן מעל ההודעה (זהו הסמל השביעי).
4. בחר גודל.
