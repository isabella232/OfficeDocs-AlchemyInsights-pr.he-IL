# <a name="update-credit-card-information"></a>עדכון פרטי כרטיס האשראי

1. במרכז הניהול של Microsoft 365, עבור לדף **חיוב** \> [שיטות תשלום](https://go.microsoft.com/fwlink/p/?linkid=2018806).

    > [!NOTE]
    > אם אתה נתקל בלולאה ששולחת אותך בחזרה לדף הבית של הניהול לפני השלמת העדכונים שלך, נסה את מצב Inprivate או גלישה בסתר.
  
2. בחר את כרטיס האשראי שברצונך לעדכן.

    > [!NOTE]
    > אם מנהל מערכת אחר הוסיף את שיטת התשלום, לא תוכל לעדכן את פרטי כרטיס האשראי או חשבון הבנק עבורו. אם תוקפו של כרטיס האשראי עומד לפוג ומנהל המערכת האחר אינו יכול לעדכן אותו, באפשרותך למנוע את הפרעת השירות על-ידי הוספת כרטיס אשראי אחר או חשבון בנק חדש עבור המנוי על-ידי בחירה באפשרות **הוסף שיטת תשלום**.
  
3. בחלונית **פרטי שיטת תשלום**, בחר **ערוך**.

4. בצע את העדכונים ולאחר מכן בחר **שמור**.

5. כדי להסיר שיטת תשלום זו, בחר **מחק**.

**פתרון בעיות**

אם אתה נתקל בבעיות בטעינת הדף, נסה לפתוח את דף האינטרנט בדפדפן אחר, או נסה לפתוח את הדף בחלון דפדפן פרטי, כגון חלון InPrivate ב- Microsoft Edge או חלון גלישה בסתר ב- Google Chrome. 

לקבלת מידע נוסף, ראה [שנה שיטות תשלום](/microsoft-365/commerce/billing-and-payments/manage-payment-methods).
