# <a name="project-to-a-pc"></a>Project למחשב

במכשיר היעד שלך (הקרנה אל), חפש את "Projection הגדרות" כדי לפתוח את דף הגדרות של **Projecting למחשב זה**. לאחר מכן ודא כי:
- "חלק Windows ומכשירי Android יכולים לפרוייקט למחשב זה כאשר אתה אומר שזה בסדר" תפריט נפתח מוגדר **כ'תמיד כבוי'.**
- התפריט הנפתח 'בקש פרוייקט למחשב זה' מוגדר לכל **פעם שנדרש חיבור.**
- התפריט הנפתח 'דרוש מספר זיהוי אישי עבור שיוך' מוגדר כ'לעולם **לא'.**

במכשיר היעד, הפעל את **התחברות על-ידי** עבור אל **התחל** וחפש את "התחברות".

לאחר מכן, במכשיר המקור שאתה מנסה לפרוייקט ממנו:

1. הקש **Windows + A כדי לפתוח** את מרכז הפעולות.
2. לחץ **על התחברות**.
3. לחץ על המכשיר שברצונך לפרוייקט את המסך שלו.

![Project למחשב](media/project-to-a-pc.png)

לאחר השלבים לעיל, התקן היעד אמור להציג את המסך של התקן המקור כאילו הוא צג משני.
