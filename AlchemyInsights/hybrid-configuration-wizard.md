# <a name="hybrid-configuration-wizard"></a>אשף קביעת התצורה ההיברידית

אשף קביעת התצורה ההיברידית הוא יישום שיעזור לך להגדיר פריסה היברידית חדשה בין הארגון Exchange המקומי Microsoft 365. אם כבר הגדרת פריסה היברידית, האשף יעזור לך לעדכן את הגדרות הפריסה ההיברידית שלך. לקבלת מידע נוסף, עיין בפרטים הבאים:
  
- [הפעלת אשף קביעת התצורה ההיברידית](https://technet.microsoft.com/library/mt595788%28v=exchg.150%29.aspx)

- [אשף קביעת התצורה ההיברידית](https://technet.microsoft.com/library/hh529921%28v=exchg.150%29.aspx)

- [שאלות נפוצות של אשף קביעת התצורה ההיברידית](https://technet.microsoft.com/library/mt488940%28v=exchg.150%29.aspx)
